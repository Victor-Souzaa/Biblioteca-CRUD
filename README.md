# Biblioteca - Sistema de Login

Este é um sistema de login básico para uma biblioteca, com validação de campos e exibição dinâmica de senha. O layout da página foi desenvolvido com HTML e CSS, e as funcionalidades de validação são gerenciadas através de JavaScript.

## Funcionalidades

O projeto inclui as seguintes funcionalidades:

1. *Validação de Campos:*
   - Validação de email com verificação de formato.
   - Validação para senha não vazia.
   - Habilitação de botões de login e recuperação de senha apenas quando os campos forem preenchidos corretamente.

2. *Exibição de Senha:*
   - Função para alternar entre exibir e ocultar o campo de senha.
   - Ícone interativo que muda ao exibir/ocultar a senha.

3. *Interface Responsiva:*
   - Layout responsivo e adaptável para diferentes tamanhos de tela.
   - Formulário centralizado com design moderno e clean.

bash

└── README.md        # Arquivo de documentação (você está aqui)


## Como Executar o Projeto

1. *Clonar o Repositório:*

   Para rodar este projeto localmente, clone o repositório:

   bash
   git clone https://github.com/Victor-Souzaa/Biblioteca
   

2. *Abrir o arquivo HTML:*

   Navegue até a pasta do projeto e abra o arquivo index.html em seu navegador.

3. *Executar o JavaScript:*

   Certifique-se de que o arquivo index.js está carregado corretamente, pois ele é responsável pelas validações de campos e funcionalidades de exibição de senha.
